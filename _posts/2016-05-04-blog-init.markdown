It's been a while since I first thought of creating my GitHub homepage and blog. Can't believe I finally made it happen.

This is also my first time using [Jekyll](https://jekyllrb.com/) to create a website. Apart from a few CSS-related headaches, the journey to completion was smooth. The markdown to HTML conversion is a joy, and blog maintenance is refreshingly easy. You could [fork](https://github.com/nihal111/nihal111.github.io/fork) this [repo](https://github.com/nihal111/nihal111.github.io) and play around with it.

I've made a [projects](/projects) section which I'll keep updating with my latest developments and contributions. I plan to use this blog to write about my experiences and other interesting stuff I come across.
